内部类：就是在一个类中定义一个类。举例：在一个类A的内部定义一个类B，类B就被成为内部类

成员内部类

按照内部类在类中定义的位置不同，可以分为如下两种形式

  • 在类的成员位置：成员内部类
  • 在类的局部位置：局部内部类

成员内部类，外界如何创建对象使用呢？

  • 格式：外部类名.内部类名 对象名 = 外部类对象.内部类对象；
  • 范例：Outer.Inner oi = new Outer().new Inner();
